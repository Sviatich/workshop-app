# 🧰 Конфигуратор упаковки на Laravel

Проект — конфигуратор картонных коробок с расчётом стоимости, выбором конструкций, оформлением заказа без регистрации и дальнейшим развитием (блог, оплата, админка).

---

## 🔥 Основные цели MVP

- Одна форма конфигурации коробки на главной странице
- Расчёт стоимости коробки и дополнительных опций на лету
- Добавление нескольких позиций в корзину
- Оформление заказа без регистрации
- Уникальная страница после оформления
- Возможность рассчёта доставки (СДЭК)
- Простая архитектура и возможность масштабирования

---

## 🧱 Архитектура

### 📦 Конструкция

- `slug`, `name`, `description`, `preview_image`, `animation`
- `preview_image` — статичная `.jpg`
- `animation` — `.webm`, запускается при наведении (через `<video>` с `poster`)

### 📐 Размеры

- Таблица `construction_sizes`
- Каждая конструкция имеет список размеров в наличии
- Используется для подбора стандартного размера при вводе произвольного (±50 мм)

### ⚙️ Расчёт

- Выполняется на бэке
- Каждый тип конструкции имеет свой калькулятор (отдельный класс)
- Возвращается:
  - Цена за штуку
  - Цена за тираж
  - Вес
  - Объём
  - Поиск указанного размера выбранной конструкции в списке размеров в наличии. Если нету то + 5000 руб к стоимости заказа за изготовление штампа. 
  - Оформление короба и его стоимость

### 🧾 Оформление короба

- Жёстко заданные опции: логотип, обечайка, окно, полноцвет
- Передаются с формы
- Обрабатываются через `DecorationPriceCalculator`
- Складываются к итоговой цене

---

## 🛒 Корзина и заказ

- Каждая позиция сохраняется в корзине на фронте (localStorage / sessionStorage)
- Заказ отправляется на сервер
- Сохраняется как `orders` + `order_items` + файлы
- Генерируется уникальная страница
- Поддерживается загрузка файлов и расчёт доставки
- В корзине есть выбор платильщика (физ.лицо и юр.лицо). Для юр.лица добавляется поле ИНН

---

## 💻 Форма

- Состоит из 3 блоков:
  1. Выбор конструкции
  2. Заполнение параметров (Д × Ш × В, цвет, профиль, тираж, (можно выбрать предлагаемый похожий размер из наличия))
  3. Опции оформления (чекбоксы + размер + файл)

- При вводе размеров система предлагает **похожие из наличия**
- Клик по предложенному → подстановка значений
- Если совпадений нет — выводится фраза о нестандартности

---

## 📦 Калькуляторы конструкций

- Расположены в `app/Services/Calculators/`
- Один файл = один класс:
  - `Fefco0427Calculator.php`
  - `LastochkinHvostCalculator.php`
- Используют вспомогательные классы:
  - `BoxPriceHelper` — цена листа по цвету
  - `DecorationPriceCalculator` — стоимость опций

## 🛠 Админка (позже)

- Laravel Starter Kit + Filament
- Разделы:
  - Конструкции (название, описание, изображения, анимации)

---

## 📌 Будущие доработки

- Подключение оплаты (например, ЮКасса)
- Блог. (Вывод постов, создание, удаление, редактирование)
- Каталог конструкций с выводом всех возможных размеров. При клике на размер можно перейти на форму с автоподстановкой размера в поля (Длина * Ширина * Высота)
- Оформленный заказ отправляется в Битрикс24 и еще формируется письмо на почту. Статус заказа из Битрикс24 синхронизируется с сервисом

---

## 📣 Принципы кода

- Простая и читаемая архитектура
- Логика расчёта — в отдельных классах
- Настройки и коэффициенты — в базе (таблица `settings`)
- Повторы — в helpers
- Без DI, без лишних абстракций
- Быстрое добавление конструкций и формул

---

## ⚠️ Примечания

- Регистрация пользователей отсутствует
- Один заказ = данные клиента + массив позиций
- Все действия с конфигурацией и корзиной — на фронте
- Уникальная страница заказа после оформления
